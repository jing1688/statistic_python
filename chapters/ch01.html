    Chapter 01

    Google Colab 基礎

    認識 Colab 操作環境、檔案管理方式，並快速複習 Python 基礎語法。

    1. 什麼是 Google Colab？

    Google Colab（全名 Colaboratory）是 Google 提供的免費雲端 Python 開發環境。

    主要特色

    • 不需安裝軟體，打開瀏覽器就能撰寫與執行 Python。
    • 程式碼在 Google 的雲端伺服器執行，不占用自己電腦的主要運算資源。
    • 可依當下額度與資源狀況使用 GPU 或 TPU。
    • Notebook 可以儲存至 Google Drive，方便存取與分享。
    開啟方式：前往 Google Colab，登入 Google 帳號即可開始使用。

    2. Colab 介面介紹

    Colab 的筆記本（Notebook）由一個個「儲存格」（Cell）組成，主要分成兩種：

    • 程式碼儲存格（Code Cell）：撰寫並執行 Python 程式碼。
    • 文字儲存格（Text Cell）：撰寫說明文字，並支援 Markdown 語法。

    執行儲存格

    • 點擊儲存格左邊的播放按鈕。
    • Shift + Enter：執行並移動至下一格。
    • Ctrl + Enter：執行但停留在目前儲存格。

    3. 檔案上傳與下載

    使用 google.colab.files 模組，可以在自己的電腦與 Colab 執行環境之間傳送檔案。

    上傳檔案

    from google.colab import files

uploaded = files.upload()

    執行後會跳出檔案選擇視窗，可以上傳 CSV、Excel 或其他資料檔案。

    下載檔案

    from google.colab import files

files.download("example.csv")

    這段程式會將 Colab 執行環境中的 example.csv 下載到自己的電腦。

    注意：直接上傳的檔案只會暫存在目前的 Colab 執行階段中。執行階段重啟、逾時或被刪除後，檔案可能消失；需要長期保存時，建議搭配 Google Drive。

    4. 掛載 Google Drive

    若資料存放在 Google 雲端硬碟，可以先掛載 Drive，再讓 Colab 直接透過路徑讀取資料。

    from google.colab import drive

drive.mount("/content/drive")

    授權完成後，可以使用下列路徑存取雲端硬碟中的檔案：

    path = "/content/drive/MyDrive/data.csv"
    路徑提醒：MyDrive 是 Google Drive 的根目錄名稱，後面的資料夾與檔名必須依照自己的實際位置修改。

    5. Python 快速複習

    變數與資料型態

    name = "Alice"       # 字串 str
age = 20             # 整數 int
height = 165.5       # 浮點數 float
is_student = True    # 布林值 bool

    串列 List

    scores = [85, 90, 78, 92]

print(scores[0])     # 85，索引從 0 開始
print(scores[-1])    # 92，-1 表示最後一筆

scores.append(100)
print(scores)        # [85, 90, 78, 92, 100]
print(len(scores))   # 5

    迴圈 for loop

    for score in scores:
    print(score)

    條件判斷 if / else

    for score in scores:
    if score >= 90:
        print(score, "優異")
    else:
        print(score, "普通")

    函式 function

    def add(a, b):
    return a + b

result = add(3, 5)
print(result)   # 8

    字典 dict

    student = {
    "name": "Alice",
    "age": 20,
    "score": 90
}

print(student["name"])   # Alice

    本章重點整理

    • Colab 是以瀏覽器操作的雲端 Python 開發環境。
    • google.colab.files 可以上傳與下載檔案。
    • google.colab.drive 可以掛載 Google Drive。
    • 本章複習了變數、串列、迴圈、條件判斷、函式與字典。

    練習題

    1. 在 Colab 建立一個新的 Notebook，並成功上傳一個檔案。
    2. 掛載自己的 Google Drive，並印出其中任一檔案的完整路徑。
    3. 建立一個包含五個整數的串列，使用迴圈判斷每個數字是奇數還是偶數。
