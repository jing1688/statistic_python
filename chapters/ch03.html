    Chapter 03

    盒鬚圖、IQR 與離群值

    使用盒鬚圖觀察資料分布，並利用四分位距判斷可能的離群值

    本章學習目標

    • 理解四分位數、四分位距與盒鬚圖的基本概念。
    • 能使用 Python 繪製盒鬚圖。
    • 能利用 IQR 方法找出可能的離群值。
    • 能解讀盒鬚圖中的中位數、分布與離群值。

    1. 四分位數

    將資料由小到大排列後，可以利用四分位數將資料分成四個部分。

    • 第一四分位數 Q1： 約有 25% 的資料小於或等於此數值。
    • 第二四分位數 Q2： 即中位數，約有 50% 的資料小於或等於此數值。
    • 第三四分位數 Q3： 約有 75% 的資料小於或等於此數值。 
    import numpy as np

scores = [52, 60, 63, 65, 68, 70, 72, 75, 78, 80, 85, 95]

q1 = np.percentile(scores, 25)
q2 = np.percentile(scores, 50)
q3 = np.percentile(scores, 75)

print("Q1：", q1)
print("Q2：", q2)
print("Q3：", q3)
    四分位數能協助我們了解資料的位置與分布情況，而且不像平均數那麼容易受到極端值影響。

    2. 四分位距 IQR

    四分位距的英文為 Interquartile Range，縮寫為 IQR。

    IQR 用來衡量資料中間 50% 的分散程度，計算公式如下：

    IQR = Q3 − Q1 
    import numpy as np

scores = [52, 60, 63, 65, 68, 70, 72, 75, 78, 80, 85, 95]

q1 = np.percentile(scores, 25)
q3 = np.percentile(scores, 75)

iqr = q3 - q1

print("Q1：", q1)
print("Q3：", q3)
print("IQR：", iqr)

    IQR 越大，表示資料中間一半的數值越分散；IQR 越小，表示資料較集中。

    3. 使用 IQR 判斷離群值

    常見的離群值判斷方式，是利用 Q1、Q3 與 IQR 建立上下界。

    下界 = Q1 − 1.5 × IQR
    上界 = Q3 + 1.5 × IQR

    小於下界或大於上界的觀察值，通常會被視為可能的離群值。 

    import numpy as np

scores = [52, 60, 63, 65, 68, 70, 72, 75, 78, 80, 85, 95, 150]

q1 = np.percentile(scores, 25)
q3 = np.percentile(scores, 75)

iqr = q3 - q1

lower_bound = q1 - 1.5 * iqr
upper_bound = q3 + 1.5 * iqr

print("Q1：", q1)
print("Q3：", q3)
print("IQR：", iqr)
print("下界：", lower_bound)
print("上界：", upper_bound)

    接著可以找出超出上下界的資料：

    outliers = [
    value
    for value in scores
    if value < lower_bound or value > upper_bound
]

print("離群值：", outliers)
    離群值不一定代表資料錯誤。它可能是真實但罕見的現象，因此在刪除離群值之前，應先了解資料背景。

    4. 盒鬚圖的結構

    盒鬚圖又稱為箱型圖或 Boxplot，可以用簡潔的方式呈現資料分布。

    • 盒子的下緣通常代表 Q1。
    • 盒子中央的線代表中位數 Q2。
    • 盒子的上緣通常代表 Q3。
    • 盒子的高度代表 IQR。
    • 鬚線延伸至非離群值中的最小值與最大值。
    • 鬚線以外的點通常表示可能的離群值。
    盒鬚圖適合快速比較不同群組的集中位置、分散程度與離群值。

    5. 使用 Matplotlib 繪製盒鬚圖

    import matplotlib.pyplot as plt

scores = [52, 60, 63, 65, 68, 70, 72, 75, 78, 80, 85, 95, 150]

plt.boxplot(scores)

plt.title("Boxplot of Scores")
plt.ylabel("Score")

plt.show()

    圖中的盒子表示 Q1 到 Q3，中間的線為中位數，單獨出現在鬚線外的點則可能是離群值。

    水平盒鬚圖

    import matplotlib.pyplot as plt

scores = [52, 60, 63, 65, 68, 70, 72, 75, 78, 80, 85, 95, 150]

plt.boxplot(
    scores,
    vert=False
)

plt.title("Horizontal Boxplot")
plt.xlabel("Score")

plt.show()

    將 vert=False 設定為水平盒鬚圖，可讓較長的數值標籤更容易閱讀。

    6. 使用 Pandas 繪製盒鬚圖

    import pandas as pd
import matplotlib.pyplot as plt

df = pd.DataFrame({
    "score": [52, 60, 63, 65, 68, 70, 72, 75, 78, 80, 85, 95, 150]
})

df.boxplot(column="score")

plt.title("Boxplot of Scores")
plt.ylabel("Score")

plt.show()

    當資料已經儲存在 DataFrame 中時，可以直接使用 boxplot() 繪圖。

    7. 比較不同群組的盒鬚圖

    盒鬚圖常用來比較不同類別或群組的數值分布。 

    import pandas as pd
import matplotlib.pyplot as plt

df = pd.DataFrame({
    "class": [
        "A", "A", "A", "A", "A",
        "B", "B", "B", "B", "B"
    ],
    "score": [
        65, 70, 72, 75, 80,
        55, 60, 62, 68, 90
    ]
})

df.boxplot(
    column="score",
    by="class"
)

plt.title("Scores by Class")
plt.suptitle("")
plt.xlabel("Class")
plt.ylabel("Score")

plt.show()

    可以從圖中比較各組的中位數、IQR、整體分散程度與離群值情況。

    9. 完整範例：找出離群值並繪圖

    import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# 建立資料
scores = [
    52, 60, 63, 65, 68, 70,
    72, 75, 78, 80, 85, 95, 150
]

# 計算四分位數
q1 = np.percentile(scores, 25)
q2 = np.percentile(scores, 50)
q3 = np.percentile(scores, 75)

# 計算 IQR
iqr = q3 - q1

# 計算離群值上下界
lower_bound = q1 - 1.5 * iqr
upper_bound = q3 + 1.5 * iqr

# 找出離群值
outliers = [
    value
    for value in scores
    if value < lower_bound or value > upper_bound
]

# 顯示計算結果
print("Q1：", q1)
print("中位數：", q2)
print("Q3：", q3)
print("IQR：", iqr)
print("離群值下界：", lower_bound)
print("離群值上界：", upper_bound)
print("離群值：", outliers)

# 繪製盒鬚圖
plt.boxplot(scores)

plt.title("Boxplot and Outliers")
plt.ylabel("Score")

plt.show()

    10. 如何解讀盒鬚圖

    觀察盒鬚圖時，可以依序檢查以下項目：

    1. 中位數位置： 判斷資料的中心位置。
    2. 盒子寬度或高度： 盒子越大，代表中間 50% 的資料越分散。
    3. 上下鬚長度： 可以觀察資料向高值或低值方向延伸的程度。
    4. 中位數是否位於盒子中央： 若偏向盒子某一側，資料可能呈現偏態。
    5. 是否存在離群點： 鬚線外的點可能是異常值或特殊觀察值。
    盒鬚圖可以顯示資料是否偏斜，但不能直接看出完整分布形狀。實務上可搭配直方圖一起判斷。

    11. 離群值應該如何處理

    找到離群值後，不應直接刪除，而應先判斷離群值產生的原因。

    • 資料輸入錯誤： 例如年齡誤輸入為 250 歲，可以修正或刪除。
    • 測量錯誤： 例如儀器故障造成異常數值。
    • 真實特殊事件： 例如極高的銷售額，可能是促銷活動造成。
    • 研究重點： 某些異常事件本身可能正是分析的重要對象。
    是否刪除離群值，應根據研究目的、資料來源與專業背景判斷，而不能只依照公式決定。

    12. 常見錯誤

    • 看到盒鬚圖外的點，就直接認定資料錯誤。
    • 沒有先處理缺失值，就直接計算四分位數。
    • 不同群組的樣本數差異很大，卻只看盒鬚圖比較。
    • 只看平均數，忽略中位數與資料分布。
    • 將 Python 中的所有極端值都直接刪除。

    若資料中包含缺失值，可以先使用 Pandas 的 dropna()： 

    clean_scores = df["score"].dropna()

q1 = clean_scores.quantile(0.25)
q3 = clean_scores.quantile(0.75)

iqr = q3 - q1

    本章重點整理

    • Q1、Q2、Q3 將排序後的資料分成四個部分。
    • IQR 等於 Q3 減去 Q1，表示中間 50% 資料的分散程度。
    • 離群值下界為 Q1 − 1.5 × IQR。
    • 離群值上界為 Q3 + 1.5 × IQR。
    • 盒鬚圖可以呈現中位數、四分位數、分散程度與可能的離群值。
    • 離群值不一定是錯誤資料，刪除前應先確認原因。
