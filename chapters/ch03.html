    Chapter 03

    盒鬚圖與相關性分析

    使用盒鬚圖檢查資料分布與離群值，並分析變數之間的相關性

    本章學習目標

    • 理解盒鬚圖的基本結構與統計意義。
    • 了解四分位數與四分位距 IQR。
    • 能使用 IQR 方法判斷可能的離群值。
    • 能使用 Python 繪製並美化盒鬚圖。

    1. 盒鬚圖的核心觀念

    盒鬚圖（Boxplot）是一種用來呈現資料分布的統計圖表，可以快速觀察資料的中位數、 四分位數、分散程度與可能的離群值。

    • Q1：第一四分位數，約有 25% 的資料小於或等於此值。
    • Q2：第二四分位數，也就是中位數。
    • Q3：第三四分位數，約有 75% 的資料小於或等於此值。
    • IQR：四分位距，計算方式為 Q3 − Q1。
    盒子的左側與右側分別代表 Q1 與 Q3，盒子中的線代表中位數， 鬚線外單獨顯示的點通常表示可能的離群值。

    2. 使用 IQR 判斷離群值

    常見的離群值判斷方式，是利用四分位距建立上下界：

    IQR = Q3 − Q1
    下界 = Q1 − 1.5 × IQR
    上界 = Q3 + 1.5 × IQR

    若某個數值小於下界，或大於上界，就可以將其視為一個可能的離群值。 

    import numpy as np

scores = [
    52, 60, 63, 65, 68, 70, 72,
    75, 78, 80, 85, 95, 150
]

q1 = np.percentile(scores, 25)
q3 = np.percentile(scores, 75)

iqr = q3 - q1

lower_bound = q1 - 1.5 * iqr
upper_bound = q3 + 1.5 * iqr

outliers = [
    score
    for score in scores
    if score < lower_bound or score > upper_bound
]

print("Q1：", q1)
print("Q3：", q3)
print("IQR：", iqr)
print("下界：", lower_bound)
print("上界：", upper_bound)
print("離群值：", outliers)
    離群值不一定是錯誤資料。它可能是真實但少見的觀察值，因此刪除前應先了解資料背景。

    3. 使用 Python 繪製盒鬚圖

    上述所有的圖表皆為基本樣式與配色，但為了能進一步增加可視性與美觀，程式也有很多設計模組去供使用。 以下使用 Matplotlib 繪製水平盒鬚圖之簡易範例，並調整畫布比例、色彩、線條與離群值樣式。 

    import matplotlib.pyplot as plt

# 建立分數資料，其中 150 可能會被判斷為離群值
scores = [
    52, 60, 63, 65, 68, 70, 72,
    75, 78, 80, 85, 95, 150
]

# 建立寬而扁的長方形畫布
# 第一個數字代表寬度，第二個數字代表高度
plt.figure(figsize=(9, 2.8))

# 繪製盒鬚圖
plt.boxplot(
    scores,

    # 將盒鬚圖改為水平方向
    vert=False,

    # 允許設定盒子的填充顏色
    patch_artist=True,

    # 設定盒子的寬度
    widths=0.5,

    # 設定盒子的填充色、邊框色與線條粗細
    boxprops=dict(
        facecolor="#CFE8FF",
        edgecolor="#5B8DB8",
        linewidth=1.8
    ),

    # 設定中位數線條的顏色與粗細
    medianprops=dict(
        color="#D9534F",
        linewidth=2
    ),

    # 設定鬚線的顏色與粗細
    whiskerprops=dict(
        color="#5B8DB8",
        linewidth=1.5
    ),

    # 設定鬚線末端短線的顏色與粗細
    capprops=dict(
        color="#5B8DB8",
        linewidth=1.5
    ),

    # 設定離群值的形狀、顏色與大小
    flierprops=dict(
        marker="o",
        markerfacecolor="#FFB6B9",
        markeredgecolor="#D9534F",
        markersize=7
    )
)

# 設定圖表標題
plt.title("Boxplot of Scores", fontsize=14)

# 設定 X 軸名稱
plt.xlabel("Score")

# 加入 X 軸方向的虛線格線，幫助讀者判讀數值位置
plt.grid(
    axis="x",
    linestyle="--",
    alpha=0.4
)

# 自動調整圖表邊界，避免文字被裁切
plt.tight_layout()

# 顯示圖表
plt.show()

    執行結果

    4. 圖表解讀

    圖中的淺藍色盒子代表 Q1 到 Q3，也就是中間 50% 的資料範圍； 盒子中央的紅色線條代表中位數。

    左右兩側的鬚線會延伸至仍位於正常範圍內的最小值與最大值。 位於右側且與盒子分離的粉紅色圓點，代表數值 150， 因為它超過 IQR 方法所計算出的上界，因此被標示為可能的離群值。

    盒鬚圖能幫助我們快速了解資料的中心位置、分散程度、偏斜方向與離群值， 但無法完整呈現資料的分布形狀，因此也可以搭配直方圖一起分析。

    5. 提升圖表美觀與可視性

    Matplotlib 的預設圖表可以透過調整畫布尺寸、圖表方向、填充顏色、 線條粗細、字體大小、格線與離群值樣式，提升圖表的美觀程度與可讀性。

    例如，單一組資料的盒鬚圖通常適合使用水平排列， 並採用較寬、較低的長方形畫布；淺色填充可以減少視覺負擔， 而中位數與離群值則可使用較醒目的顏色，讓重要資訊更容易辨識。

    圖表美化不只是改變顏色，而是要讓讀者更快找到重點。 良好的圖表設計應兼顧美觀、清楚與統計資訊的正確呈現。

    6. 共變異數（Covariance）

    共變異數用來觀察兩個變數是否傾向一起變動。 若兩個變數經常往相同方向變動，共變異數通常為正； 若經常往相反方向變動，共變異數通常為負。

    • 共變異數大於 0：兩個變數大致同方向變動。
    • 共變異數小於 0：兩個變數大致反方向變動。
    • 共變異數接近 0：可能沒有明顯的線性共同變動。

    以下以讀書時間與考試成績為例：

    import numpy as np

study_hours = np.array([2, 3, 4, 5, 6, 7, 8, 9])
scores = np.array([55, 60, 65, 68, 72, 78, 82, 88])

covariance_matrix = np.cov(
    study_hours,
    scores
)

print(covariance_matrix)

    np.cov() 會回傳共變異數矩陣，其中右上角與左下角的數值， 就是讀書時間與考試成績之間的樣本共變異數。

    也可以直接取出兩個變數之間的共變異數：

    covariance = np.cov(
    study_hours,
    scores
)[0, 1]

print("共變異數：", covariance)
    共變異數會受到資料單位影響。例如將「小時」改成「分鐘」， 共變異數的數值也會改變，因此通常不適合直接比較不同資料組合的關係強弱。

    7. 相關係數（Correlation）

    相關係數將共變異數標準化，數值介於 −1 與 1 之間， 可用來描述兩個變數線性關係的方向與強度。

    • 接近 1：具有較強的正線性關係。
    • 接近 −1：具有較強的負線性關係。
    • 接近 0：線性關係較弱。

    Pearson 相關係數

    import numpy as np

study_hours = np.array([2, 3, 4, 5, 6, 7, 8, 9])
scores = np.array([55, 60, 65, 68, 72, 78, 82, 88])

correlation_matrix = np.corrcoef(
    study_hours,
    scores
)

print(correlation_matrix)

    若只需要兩個變數之間的相關係數，可以寫成：

    correlation = np.corrcoef(
    study_hours,
    scores
)[0, 1]

print("Pearson 相關係數：", correlation)

    np.corrcoef() 會回傳相關矩陣。矩陣對角線固定為 1， 而右上角與左下角則是兩個變數之間的相關係數。

    使用 Pandas 計算相關矩陣

    import pandas as pd

df = pd.DataFrame({
    "study_hours": [2, 3, 4, 5, 6, 7, 8, 9],
    "attendance": [65, 70, 72, 75, 80, 85, 88, 92],
    "score": [55, 60, 65, 68, 72, 78, 82, 88]
})

correlation_matrix = df.corr()

print(correlation_matrix)
    相關係數只能表示變數之間的關聯程度，不能直接證明因果關係。 即使讀書時間與成績高度相關，也不能只靠相關係數認定所有成績差異都由讀書時間造成。

    (補充) 散佈圖與線性趨勢線

    散佈圖可以呈現兩個數值變數之間的關係。 若資料點大致沿著一條直線排列，代表兩個變數可能具有線性關係。

    基本散佈圖

    import matplotlib.pyplot as plt

study_hours = [2, 3, 4, 5, 6, 7, 8, 9]
scores = [55, 60, 65, 68, 72, 78, 82, 88]

plt.figure(figsize=(8, 5))

plt.scatter(
    study_hours,
    scores
)

plt.title("Study Hours and Scores")
plt.xlabel("Study Hours")
plt.ylabel("Score")

plt.grid(
    linestyle="--",
    alpha=0.35
)

plt.tight_layout()
plt.show()

    加入線性迴歸趨勢線

    import numpy as np
import matplotlib.pyplot as plt

study_hours = np.array([2, 3, 4, 5, 6, 7, 8, 9])
scores = np.array([55, 60, 65, 68, 72, 78, 82, 88])

# 建立一階線性迴歸模型
slope, intercept = np.polyfit(
    study_hours,
    scores,
    1
)

# 計算趨勢線上的預測值
trend_scores = (
    slope * study_hours + intercept
)

plt.figure(figsize=(8, 5))

# 繪製原始資料點
plt.scatter(
    study_hours,
    scores,
    s=65,
    color="#9CC9F5",
    edgecolor="#4F81BD",
    linewidth=1.2,
    label="Observed data"
)

# 繪製線性趨勢線
plt.plot(
    study_hours,
    trend_scores,
    color="#E07A5F",
    linewidth=2.2,
    label="Linear trend"
)

plt.title(
    "Study Hours and Scores",
    fontsize=14
)

plt.xlabel("Study Hours")
plt.ylabel("Score")

plt.grid(
    linestyle="--",
    alpha=0.35
)

plt.legend()
plt.tight_layout()
plt.show()

    np.polyfit(x, y, 1) 會建立一階線性模型， 並回傳斜率與截距。趨勢線的方程式可寫成：

    預測值 = 斜率 × x + 截距

    也可以印出迴歸方程式：

    print("斜率：", slope)
print("截距：", intercept)

print(
    f"迴歸方程式：score = "
    f"{slope:.2f} × study_hours + "
    f"{intercept:.2f}"
)

    趨勢線解讀

    若趨勢線由左下往右上延伸，表示兩個變數具有正向關係； 若由左上往右下延伸，則表示兩個變數具有負向關係。

    趨勢線可以協助觀察資料的整體方向，但不代表所有資料點都會落在線上。 分析時仍應搭配相關係數、離群值與實際資料背景進行判斷。

    練習題

    1. 請計算資料中的 Q1、Q3 與 IQR。
    2. 請利用 IQR 方法找出資料中的可能離群值。
    3. 請使用 Matplotlib 繪製一張水平盒鬚圖。
    4. 請將盒子的顏色改成其他淺色，並調整中位數線條。
    5. 請說明為什麼找到離群值後不能直接刪除。

    本章重點整理

    • 盒鬚圖可呈現中位數、四分位數、分散程度與可能的離群值。
    • IQR 等於 Q3 減去 Q1。
    • IQR 方法利用上下界判斷可能的離群值。
    • 離群值不一定是錯誤，刪除前應先確認形成原因。
    • 適當調整畫布、配色、線條與格線，可以提升圖表可讀性。
