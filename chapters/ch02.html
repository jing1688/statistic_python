    資料讀取與繪圖

    使用 Pandas 讀取常見資料格式，計算平均數、中位數、眾數與幾何平均數， 並以 Matplotlib、Pandas 與 Seaborn 建立圖表。

    1. 讀取本機上傳的檔案

    在 Google Colab 中，可以先透過 google.colab.files 上傳檔案，再使用 Pandas 讀取。 

    from google.colab import files
import pandas as pd

uploaded = files.upload()

df = pd.read_csv("data.csv")

df.head()
    提醒： 程式碼中的 data.csv 必須與實際上傳的檔名完全相同， 包含副檔名以及英文字母大小寫。

    2. 使用 Pandas 讀取不同格式的檔案

    讀取 CSV

    import pandas as pd

df = pd.read_csv(
    "data.csv",
    encoding="utf-8",
    sep=","
)

df.head()
    若中文 CSV 出現亂碼，可以依照檔案的實際編碼， 嘗試使用 utf-8-sig、 big5 或 cp950。

    讀取 Excel

    df = pd.read_excel(
    "data.xlsx",
    sheet_name="Sheet1"
)

df.head()

    sheet_name 可以填入工作表名稱， 也可以用數字指定工作表的位置。 

    # 讀取第一個工作表
df = pd.read_excel(
    "data.xlsx",
    sheet_name=0
)
    sheet_name=0 代表第一個工作表， sheet_name=1 則代表第二個工作表。

    讀取 JSON

    df = pd.read_json("data.json")

df.head()

    3. 從 Google Drive 讀取資料

    若檔案已經存放在 Google Drive， 可以先將 Drive 掛載到 Colab，再使用檔案路徑讀取資料。 

    from google.colab import drive

drive.mount("/content/drive")

df.head()

    實務上建議先在 Colab 左側的檔案瀏覽器中找到檔案， 再複製完整路徑，較不容易輸入錯誤。

    4. 基本敘述統計量

    在繪製圖表之前，可以先使用基本統計量了解資料的中心位置與分布情形。 常見的集中趨勢統計量包括：

    • 算術平均數 Arithmetic Mean
    • 幾何平均數 Geometric Mean
    • 眾數 Mode
    • 中位數 Median

    算術平均數 Arithmetic Mean

    算術平均數是將所有資料加總後，再除以資料筆數。

    假設有以下五筆資料：

    data = [10, 20, 30, 40, 50]

    其平均數計算方式為：

    平均數 = (10 + 20 + 30 + 40 + 50) ÷ 5 = 30

    不使用 mean() 計算平均數

    可以先使用 sum() 計算總和， 再使用 len() 計算資料筆數。 

    data = [10, 20, 30, 40, 50]

total = sum(data)
number_of_data = len(data)

average = total / number_of_data

print("資料總和：", total)
print("資料筆數：", number_of_data)
print("平均數：", average)

    執行結果：

    資料總和： 150
資料筆數： 5
平均數： 30.0

    使用迴圈自行計算總和

    若希望進一步理解加總的運作方式， 也可以使用 for 迴圈逐筆累加。 

    data = [10, 20, 30, 40, 50]

total = 0

for value in data:
    total = total + value

average = total / len(data)

print("平均數：", average)
    計算概念： 平均數等於所有資料的總和除以資料筆數。

    使用 NumPy 的 mean()

    了解平均數的計算概念後， 可以使用 NumPy 的 mean() 快速完成計算。 

    import numpy as np

data = [10, 20, 30, 40, 50]

average = np.mean(data)

print("平均數：", average)

    使用 Pandas 的 mean()

    若資料已經存放在 Pandas DataFrame 中， 可以對指定欄位使用 .mean()。 

    average = df["column_name"].mean()

print("平均數：", average)

    幾何平均數 Geometric Mean

    幾何平均數是先將所有數值相乘， 再對乘積開資料筆數次方根。

    幾何平均數常用於成長率、投資報酬率、 比例變化以及具有連乘關係的資料。

    假設資料為 2、8、4， 幾何平均數的計算方式為：

    幾何平均數 = (2 × 8 × 4)1/3 = 4

    自行計算幾何平均數

    import math

data = [2, 8, 4]

product = math.prod(data)

geometric_mean = product ** (1 / len(data))

print("資料乘積：", product)
print("幾何平均數：", geometric_mean)

    執行結果：

    資料乘積： 64
幾何平均數： 4.0

    使用 SciPy 的 gmean()

    from scipy.stats import gmean

data = [2, 8, 4]

geometric_mean = gmean(data)

print("幾何平均數：", geometric_mean)
    注意： 一般的幾何平均數適用於正數資料。 若資料包含 0 或負數，應先確認該資料是否適合使用幾何平均數。

    眾數 Mode

    眾數是資料中出現次數最多的數值。

    一組資料可能只有一個眾數，也可能同時有多個眾數。

    例如：

    data = [1, 2, 2, 3, 3, 3, 4]

    數字 3 出現三次， 是出現次數最多的數值，因此眾數為 3。

    使用 Counter 計算眾數

    from collections import Counter

data = [1, 2, 2, 3, 3, 3, 4]

frequency = Counter(data)

highest_frequency = max(frequency.values())

modes = [
    value
    for value, count in frequency.items()
    if count == highest_frequency
]

print("各數值出現次數：", frequency)
print("眾數：", modes)

    執行結果：

    各數值出現次數： Counter({3: 3, 2: 2, 1: 1, 4: 1})
眾數： [3]

    使用 Pandas 的 mode()

    import pandas as pd

data = pd.Series(
    [1, 2, 2, 3, 3, 3, 4]
)

modes = data.mode()

print("眾數：")
print(modes)
    Pandas 的 .mode() 會回傳 Series， 因為一組資料可能同時存在兩個以上的眾數。

    中位數 Median

    中位數是將資料由小到大排列後， 位於中央位置的數值。

    中位數的計算方式會依資料筆數為奇數或偶數而不同。

    資料筆數為奇數

    假設資料為：

    data = [7, 2, 9, 4, 5]

    將資料排序後為 [2, 4, 5, 7, 9]， 中間的數值是 5，因此中位數為 5。 

    data = [7, 2, 9, 4, 5]

sorted_data = sorted(data)

middle_index = len(sorted_data) // 2

median = sorted_data[middle_index]

print("排序後資料：", sorted_data)
print("中位數：", median)

    執行結果：

    排序後資料： [2, 4, 5, 7, 9]
中位數： 5

    資料筆數為偶數

    假設資料為：

    data = [7, 2, 9, 4, 5, 8]

    將資料排序後為 [2, 4, 5, 7, 8, 9]。

    因為資料筆數為偶數， 中位數為中間兩個數值 5 與 7 的平均：

    (5 + 7) ÷ 2 = 6 

    data = [7, 2, 9, 4, 5, 8]

sorted_data = sorted(data)

middle_index = len(sorted_data) // 2

left_value = sorted_data[middle_index - 1]
right_value = sorted_data[middle_index]

median = (left_value + right_value) / 2

print("排序後資料：", sorted_data)
print("中位數：", median)

    執行結果：

    排序後資料： [2, 4, 5, 7, 8, 9]
中位數： 6.0

    使用 NumPy 的 median()

    import numpy as np

data = [7, 2, 9, 4, 5, 8]

median = np.median(data)

print("中位數：", median)

    使用 Pandas 的 median()

    median = df["column_name"].median()

print("中位數：", median)
    平均數與中位數的差異： 平均數容易受到極端值影響。 當資料中存在特別大或特別小的數值時， 中位數通常更能代表資料的中心位置。

    綜合範例

    以下範例同時計算一組資料的平均數、中位數與眾數。 

    import pandas as pd
import numpy as np

data = pd.Series(
    [10, 12, 12, 15, 18, 20, 100]
)

mean_value = data.mean()
median_value = data.median()
mode_value = data.mode()

print("平均數：", mean_value)
print("中位數：", median_value)
print("眾數：")
print(mode_value)
    由於資料中存在極端值 100， 平均數會被向上拉高，而中位數受到的影響較小。

    5. 使用 Matplotlib 繪圖

    Matplotlib 是 Python 最常用的基礎繪圖套件之一。 

    import matplotlib.pyplot as plt
import numpy as np
    中文顯示： Colab 的 Linux 環境通常沒有 Microsoft JhengHei。 若需要中文標題，必須另外安裝中文字體， 或先使用英文標題。

    折線圖 Line Plot

    折線圖常用於觀察資料隨時間或順序的變化趨勢。 

    x = [1, 2, 3, 4, 5]
y = [10, 20, 15, 25, 30]

plt.plot(x, y)

plt.title("Line Plot Example")
plt.xlabel("X")
plt.ylabel("Y")

plt.show()

    長條圖 Bar Plot

    長條圖適合比較不同類別之間的數值差異。 

    categories = ["A", "B", "C", "D"]
values = [23, 45, 12, 36]

plt.bar(categories, values)

plt.title("Bar Plot Example")
plt.xlabel("Category")
plt.ylabel("Value")

plt.show()

    直方圖 Histogram

    直方圖可以用來觀察數值資料的分布情形， 例如資料是否集中、偏斜或具有多個高峰。 

    data = np.random.randn(1000)

plt.hist(
    data,
    bins=30
)

plt.title("Histogram Example")
plt.xlabel("Value")
plt.ylabel("Frequency")

plt.show()

    散佈圖 Scatter Plot

    散佈圖可用於觀察兩個數值變數之間的關係。 

    x = np.random.rand(50)
y = np.random.rand(50)

plt.scatter(x, y)

plt.title("Scatter Plot Example")
plt.xlabel("X")
plt.ylabel("Y")

plt.show()

    6. Pandas 內建繪圖功能

    Pandas 的 DataFrame 與 Series 提供 .plot() 方法， 適合快速檢查資料分布與變數關係。

    使用 Pandas 繪製折線圖

    df["column_name"].plot(
    kind="line"
)

plt.show()

    使用 Pandas 繪製直方圖

    df["column_name"].plot(
    kind="hist",
    bins=20
)

plt.show()

    使用 Pandas 繪製散佈圖

    df.plot(
    x="col1",
    y="col2",
    kind="scatter"
)

plt.show()
    Pandas 的繪圖功能底層仍使用 Matplotlib， 因此也能搭配 plt.title()、 plt.xlabel()、 plt.ylabel() 與 plt.show() 調整圖表。

    7. 使用 Seaborn 繪製統計圖表

    Seaborn 建立在 Matplotlib 之上， 提供較簡潔的統計視覺化語法， 並能直接搭配 Pandas DataFrame。 

    import seaborn as sns
import matplotlib.pyplot as plt

sns.set_theme(
    style="whitegrid"
)

    直方圖與密度曲線

    sns.histplot(
    data=df,
    x="column_name",
    kde=True
)

plt.show()

    kde=True 會在直方圖上加入平滑的密度曲線， 有助於觀察資料的整體分布形狀。

    依類別上色的散佈圖

    sns.scatterplot(
    data=df,
    x="col1",
    y="col2",
    hue="category_column"
)

plt.show()
    hue 的用途： 依照類別欄位自動將資料點分組並使用不同顏色呈現， 有助於觀察不同群組的分布與關係。

    本章重點整理

    • Pandas 可以讀取 CSV、Excel、JSON 等多種資料格式。
    • 讀取電腦檔案前，需要先將檔案上傳至 Colab。
    • 讀取 Google Drive 中的檔案前，需要先掛載 Drive。
    • 平均數可以使用資料總和除以資料筆數計算， 也可以使用 mean()。
    • 幾何平均數適合具有連乘關係的正數資料。
    • 眾數代表資料中出現次數最多的數值。
    • 中位數是排序後位於中間位置的數值， 較不容易受到極端值影響。
    • Matplotlib 可以繪製折線圖、長條圖、直方圖與散佈圖。
    • Pandas 的 .plot() 適合快速檢視資料。
    • Seaborn 適合建立統計圖表， 並可使用 hue 比較不同群組。

    練習題

    1. 上傳一份 CSV， 使用 Pandas 讀取並顯示前五筆資料。
    2. 建立一組數值資料， 不使用 mean() 計算平均數。
    3. 使用 NumPy 的 mean() 計算相同資料的平均數， 並比較兩種方法的結果。
    4. 建立一組正數資料， 自行計算幾何平均數。
    5. 使用 Pandas 計算一個數值欄位的平均數、 中位數與眾數。
    6. 在資料中加入一個極端值， 比較平均數與中位數的變化。
    7. 選擇一個數值欄位， 使用 Matplotlib 繪製直方圖。
    8. 使用 Seaborn 的 scatterplot()， 並以 hue 依類別欄位上色。
