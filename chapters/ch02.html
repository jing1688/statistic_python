    Chapter 02

    資料讀取與繪圖

    使用 Pandas 讀取常見資料格式，並以 Matplotlib、Pandas 與 Seaborn 建立圖表。

    1. 讀取本機上傳的檔案

    先透過 google.colab.files 上傳檔案，再使用 Pandas 讀取：

    from google.colab import files
import pandas as pd

uploaded = files.upload()

df = pd.read_csv("data.csv")
df.head()
    提醒：程式碼中的 data.csv 必須與實際上傳的檔名完全相同，包含副檔名及英文字母大小寫。

    2. 使用 Pandas 讀取不同格式的檔案

    讀取 CSV

    import pandas as pd

df = pd.read_csv("data.csv", encoding="utf-8", sep=",")
df.head()
    若中文 CSV 出現亂碼，可以依照檔案實際編碼嘗試 utf-8-sig、big5 或 cp950。

    讀取 Excel

    df = pd.read_excel("data.xlsx", sheet_name="Sheet1")
df.head()

    sheet_name 可以填入工作表名稱，也可以用數字指定位置，例如 sheet_name=0 代表第一個工作表。

    讀取 JSON

    df = pd.read_json("data.json")
df.head()

    3. 從 Google Drive 讀取資料

    from google.colab import drive
drive.mount("/content/drive")

import pandas as pd

path = "/content/drive/MyDrive/data.csv"
df = pd.read_csv(path)
df.head()

    實務上建議先在 Colab 左側的檔案瀏覽器找到檔案，再複製完整路徑，較不容易打錯。

      Chapter 02

      資料讀取與繪圖

      使用 Pandas 讀取常見資料格式，並以 Matplotlib、Pandas 與 Seaborn 建立圖表。

      實務上建議先在 Colab 左側的檔案瀏覽器找到檔案，再複製完整路徑，較不容易打錯。

      4. 使用 Matplotlib 繪圖

      Matplotlib 是 Python 最常用的基礎繪圖套件之一。

      import matplotlib.pyplot as plt
import numpy as np
      中文顯示：Colab 的 Linux 環境通常沒有 Microsoft JhengHei。若需要中文標題，必須另外安裝中文字體，或先使用英文標題。

      折線圖 Line Plot

      x = [1, 2, 3, 4, 5]
y = [10, 20, 15, 25, 30]

plt.plot(x, y)
plt.title("Line Plot Example")
plt.xlabel("X")
plt.ylabel("Y")
plt.show()

      長條圖 Bar Plot

      categories = ["A", "B", "C", "D"]
values = [23, 45, 12, 36]

plt.bar(categories, values)
plt.title("Bar Plot Example")
plt.show()

      直方圖 Histogram

      data = np.random.randn(1000)

plt.hist(data, bins=30)
plt.title("Histogram Example")
plt.show()

      散佈圖 Scatter Plot

      x = np.random.rand(50)
y = np.random.rand(50)

plt.scatter(x, y)
plt.title("Scatter Plot Example")
plt.show()

      5. Pandas 內建繪圖功能

      Pandas 的 DataFrame 與 Series 提供 .plot() 方法，適合快速檢查資料分布與變數關係。

      df["column_name"].plot(kind="line")
df["column_name"].plot(kind="hist")
df.plot(x="col1", y="col2", kind="scatter")
      Pandas 的繪圖功能底層仍使用 Matplotlib，因此也能搭配 plt.title()、plt.xlabel() 與 plt.show() 調整圖表。

      6. 使用 Seaborn 繪製統計圖表

      Seaborn 建立在 Matplotlib 之上，提供較簡潔的統計視覺化語法，並能直接搭配 Pandas DataFrame。

      import seaborn as sns
import matplotlib.pyplot as plt

sns.set_theme(style="whitegrid")

      直方圖與密度曲線

      sns.histplot(data=df, x="column_name", kde=True)
plt.show()

      依類別上色的散佈圖

      sns.scatterplot(
    data=df,
    x="col1",
    y="col2",
    hue="category_column"
)
plt.show()
      hue 的用途：依照類別欄位自動替資料點分組，有助於觀察不同群組的分布與關係。

      本章重點整理

      • Pandas 可以讀取 CSV、Excel、JSON 等多種資料格式。
      • 讀取電腦檔案前，需要先上傳至 Colab；讀取 Drive 檔案前，需要先掛載。
      • Matplotlib 可以繪製折線圖、長條圖、直方圖與散佈圖。
      • Pandas 的 .plot() 適合快速檢視資料。
      • Seaborn 適合建立統計圖表，並可使用 hue 比較不同群組。

      練習題

      1. 上傳一份 CSV，使用 Pandas 讀取並顯示前五筆資料。
      2. 選擇一個數值欄位，使用 Matplotlib 繪製直方圖。
      3. 使用 Seaborn 的 scatterplot()，並以 hue 依類別欄位上色。
