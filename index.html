    用 Python 學統計學

    從 Python 基礎、描述統計與機率開始，逐步進入抽樣、統計推論、 假設檢定與迴歸分析。現在先把網站骨架搭好，之後再逐章補上教材。

    課程介紹

    本網站採用純 HTML、CSS 與 JavaScript，可直接部署到 GitHub Pages， 不需要安裝額外框架。

    適合對象

    想用 Python 入門統計學，或需要統計程式範例的大學生與自學者。

    建議環境

    Python 3、Jupyter Notebook、NumPy、Pandas、SciPy 與 Matplotlib。
